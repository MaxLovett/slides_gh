课程沿革

专业基础 + 呼应需求

  • 48学时，2021年获批
  • 建设原因
    • 由相关专业课程基础上发展而来
    • 国家需求：理解公共政策成为理解国家所需、位列行业前列的必备条件
    • 学生需求：
      • 既有课程理论性过强、实用性欠缺
      • 校内无普及性课程
  • 建课目标
    • 普及公共政策知识
    • 培养公共政策的理解力
    • 树立正确政策观
  • 课程挑战
    • 社科知识弱 → 逻辑+方法
    • 实操要求高 → “强参与式”教学
    • 学科背景杂 → “私人定制”作业

建设思路：四个“相结合”

  • 教学理念上，注重专业化与通识化相结合
    • 系统介绍公共政策的经典视角和常见方法
    • 以引介不同层次、专业领域分析视角和具有实操性的分析方法为重点。
  • 内容选择上，注重意识训练与实际操作相结合
    1. 展示公共政策制定和实施的多面性和动态性
    2. 提升学生多角度看待公共政策的能力
  • 教学过程上，注重政策复杂性理解与社科研究规范相结合
    • 理解归纳民意之繁难、反映民心之多面，培养未来决策人才的政策意识和公民责任
    • 增强社会科学研究素质，树立正确社会观、政治观、价值观。
  • 评价方式上,注重课程作业的多元性和效果评价的动态性相结合
    • “私人定制”课程作业、多维度多种评价手段

引入电梯演讲、“宾主对”、学术宣讲

课程特色

强参与式教学

  • 特色1：教学方法创新
  • 特色2：高强度互动
  • 特色3：动态评价体系

特色1：教学方法创新“堂上宾主对”

  • “主讲解”、“宾论难”、“宾主对”三个主要环节
  • 将中国传统儒学、佛学辩论式教学方法与“翻转课堂”现代教育思维融合创立的互动型教学模式
  • 旨在最大限度调动学生学习积极性，也提供展现其独特学科优势的舞台，增强学生学习获得感，在教学方式上践行“第二个结合”

特色2：高强度互动模式

  • 引入问题 → 设立目标 → 参与式教学 → 检验评估 → 知识总结
  • “在行动中学习原理”

特色3：动态评价体系

  • 10% 来自每次课堂表现。
  • 50% 来自“课堂宾主对”表现。 每周设有延申阅读一篇（即Must-Reads部分指定内容），要求学生课下阅读。 该部分材料针对课上内容精心挑选，是对课堂知识的运用或拓展。这些内容教师将不会在正式授课中涉及，而通过“宾主对”以参与式学习方式进行互动学习。 全部或部分学生可能有机会组队进行宾主对，以及参与多轮。 对于组队宾主对，队伍成员得分相同；对于多轮参与，取各次参与中的最高分计入最后成绩。
  • 40% 为期末研究计划宣讲。 学生可选择一项公共政策，依据其内容、对象和施用领域，运用课上所学理论和方法提出分析方案。 这是一个延续整个学期的项目，具体安排参见“政策分析计划”。 在课程最后阶段，每位学生均须对自己的期末研究计划进行宣讲，时长根据课程人数决定。

附加分

  1. 基于课程所学和项目内容进行SRT、大创项目等或参与公共政策大赛
  2. 自愿报名宾主对主讲
